## 🔭 What I'm Building

### [vibe.market — Solana NFT Marketplace](https://github.com/layla-lili/nft-marketplace)

A production-grade NFT marketplace on Solana built from scratch.

- **Anchor (Rust)** smart contracts — list, buy, auction, royalty enforcement
- **Go indexer** — custom WebSocket pipeline subscribing to program logs, decoding Anchor events, writing idempotently to Postgres
- **Next.js + TypeScript** frontend with Phantom/Solflare/WalletConnect support
- Built a **custom indexer instead of The Graph** to understand the full on-chain event lifecycle

> Stack: Anchor · Go (Gin + GORM) · Next.js · PostgreSQL · Docker
