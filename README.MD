## 🔭 Projects

### [vibe.market — Solana NFT Marketplace](https://github.com/layla-lili/nft-marketplace)
Production-grade NFT marketplace on Solana, rebuilt from scratch with deliberate architecture choices.
- **Anchor (Rust)** smart contracts — fixed-price listings, auctions, royalty enforcement via on-chain Metaplex metadata
- **Custom Go indexer** — WebSocket pipeline subscribing to program logs, decoding Anchor event discriminators, writing idempotently to Postgres using tx signature as unique key. No Graph Protocol.
- **Next.js + TypeScript** frontend with Phantom, Solflare, and WalletConnect support

`Anchor` `Rust` `Go` `Gin` `GORM` `Next.js` `PostgreSQL` `Docker`

---

### [Decentralized Financial Instruments Marketplace](https://github.com/layla-lili/decentralized-marketplace-for-trading-financial-instruments)
On-chain trading platform for shares, bonds, and stablecoins on a private QBFT Ethereum network.
- **5 Solidity smart contracts** — ERC20 tokens (TRG, CLV, ROO, GOV) + Vault custody contract, deployed with Foundry
- **Real-time order matching engine** in Express.js — auto-matches BUY/SELL orders, reserves balances, handles cancellations
- **KYC + wallet-based auth**, portfolio view, price charts, deposit/withdrawal cycle with on-chain settlement via Viem

`Solidity` `Foundry` `QBFT` `Express.js` `Prisma` `PostgreSQL` `Next.js` `Viem` `Docker`

---

### [Benchy — Private Ethereum Network Dashboard & CLI](https://github.com/layla-lili/private-Ethereum-networks-dashboard-cli)
CLI tool for launching, monitoring, and stress-testing private Ethereum networks with a real-time web dashboard.
- **5-node network** (3 Geth validators + 1 Nethermind validator + 2 observers) spun up with a single command via Docker
- **5 automated test scenarios** — balance checks, continuous transfers, ERC20 deployment, transaction ordering/race conditions
- **Failure injection** — stop/restart individual nodes, observe consensus recovery in real time
- Real-time dashboard showing block height, peer connections, mempool, CPU/memory per node

`TypeScript` `Node.js` `Geth` `Nethermind` `Clique PoA` `ethers.js` `Docker` `Commander.js`
